<br>

## Dependency

[Ncurses](https://en.wikipedia.org/wiki/Ncurses) (this library is not available natively on Windows).

## To build

```
g++ *.cpp -std=c++14 -lncurses
```

## To run the binary

```
./a.out
```

## To build and run the binary

```
g++ *.cpp -std=c++14 -lncurses && ./a.out
```
